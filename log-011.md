# Log 011 - Documenting the Geometry Core

## Primitive Axioms as a Mathematical API

The current Geometry core consists of eleven primitive axioms. Rather than
reading their Lean declarations one by one, it is useful to document them
as a mathematical API. Each entry explains the meaning of the arguments,
the geometric statement, the role of the axiom, and where it first appears
in the Finlay formalization.

| Lean declaration | Informal statement | Purpose | First used |
|------------------|--------------------|----------|------------|
| `midpoint_collinear (A B M)` | `M` midpoint of `AB` ⇒ `A,M,B` are collinear. | Incidence property of a midpoint. | `FinlayStep1` |
| `collinear_swap (A B C)` | `A,B,C` collinear ⇒ `A,C,B` collinear. | Reorder collinear points. | `FinlayStep1` |
| `MidsegmentTheorem (A B C M N)` | `M` midpoint of `AB`, `N` midpoint of `AC` ⇒ `MN || BC`. | Affine theorem for triangles. | `FinlayStep1` |
| `ParallelCollinearLeft (A B C D E)` | `AB || DE` and `C∈AB` ⇒ `CB || DE`. | Move a parallel line along the same supporting line. | `FinlayStep2` |
| `ParallelSymm (A B C D)` | `AB || CD` ⇒ `CD || AB`. | Symmetry of parallelism. | `FinlayStep2` |
| `ParallelFlipRight (A B C D)` | `AB || CD` ⇒ `AB || DC`. | Reverse the orientation of one parallel line. | `FinlayStep2` |
| `ParallelogramOfParallel (A B C D)` | `AB || CD` and `AD || BC` ⇒ `ABCD` is a parallelogram. | Build a parallelogram from two pairs of opposite parallel sides. | `FinlayStep3` |
| `ParallelogramDiagonals (A B C D M)` | If `M` lies on diagonals `AC` and `BD` of parallelogram `ABCD`, then `M` is the midpoint of `AC`. | Diagonal bisection property. | `FinlayStep4` |
| `MidpointMedian (A B C M)` | `M` midpoint of `BC` ⇒ `AM` is a median of triangle `ABC`. | Connect midpoint and median. | `FinlayStep5` |
| `IntersectionOnSameLine (A G P B C D)` | Replacing one representation of a line by another collinear representation preserves the intersection relation. | Change the description of an intersection. | Auxiliary |
| `CollinearTrans (A G P D)` | `A,G,P` and `P,D,G` collinear ⇒ `A,G,D` collinear. | Transitivity of collinearity. | Auxiliary |

## Organization

| Category | Primitive axioms |
|----------|------------------|
| Collinearity | `midpoint_collinear`, `collinear_swap`, `CollinearTrans` |
| Parallelism | `MidsegmentTheorem`, `ParallelCollinearLeft`, `ParallelSymm`, `ParallelFlipRight` |
| Parallelograms | `ParallelogramOfParallel`, `ParallelogramDiagonals` |
| Triangle geometry | `MidpointMedian` |
| Incidence | `IntersectionOnSameLine` |

## Dependency Perspective

Read from top to bottom, the table almost reconstructs the logical structure
of the Finlay proof:

1. Midpoints produce collinearity.
2. Midpoints produce a midsegment.
3. The midsegment gives parallelism.
4. Parallelism creates a parallelogram.
5. The parallelogram yields a midpoint on a diagonal.
6. The midpoint becomes a median.

This makes the document both a reference manual and a dependency map for the
current Geometry core.
