# Log 011 - Geometry Core: Primitive Relations, Definitions, and Axioms

## Summary

During the reverse engineering of the Geometry library, the logical status of the basic geometric concepts was systematically re-evaluated.

The original implementation introduced several predicates as primitive members of the `Geo` structure. The objective of this experiment was to determine whether these predicates were genuinely primitive or merely convenient abbreviations of more elementary relations.

The refactoring demonstrated that several of these predicates can be replaced by ordinary Lean definitions without affecting the formal proof of Finlay's theorem.

# Results of the Refactoring

The following concepts were converted from primitive predicates into ordinary definitions:

- `IsMidpoint`
- `IsIntersection` (already definitional, retained)
- `IsMedian`
- `IsParallelogram`

As a consequence, several former axioms became ordinary theorems:

| Previous axiom | Current status |
|---------------|----------------|
| `midpoint_collinear` | theorem |
| `midpoint_congruent` | theorem |
| `MidpointMedian` | theorem |
| `ParallelogramOfParallel` | theorem |

No conceptual modification of the Finlay proof was required.

Only the interfaces of these concepts changed; the proof itself remained structurally identical.

The complete development continues to compile successfully after the refactoring.

---

# Observation

This experiment indicates that the logical kernel of the Geometry library is significantly smaller than suggested by the original interface.

Several notions that initially appeared to be primitive geometric concepts are, in fact, definitional abstractions built from more elementary relations.

The resulting organization separates the library into four clearly distinguishable levels:

1. primitive geometric relations,
2. derived geometric definitions,
3. structural properties of the primitive relations,
4. genuine geometric theorems.

This layered architecture provides a clearer conceptual model of the library and offers a cleaner foundation for future formal developments.
